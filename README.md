AWS 3-Tier Web Application with Docker

Overview

This project demonstrates the deployment of a 3-tier web application on AWS using Docker containers.

Architecture

Tier 1: Frontend (HTML/Nginx)
Tier 2: Backend (Python Flask)
Tier 3: Database (AWS RDS MySQL)
Technologies Used

AWS EC2
AWS RDS MySQL
Docker
Docker Compose
Python Flask
Nginx
HTML
Linux
Features

Frontend hosted using Nginx
Backend API developed using Flask
Database hosted on AWS RDS MySQL
Containerized deployment using Docker
Project Workflow

Launch AWS EC2 instance.
Install Docker.
Create frontend and backend applications.
Build Docker images.
Configure AWS RDS MySQL.
Run containers.
Access the application using the EC2 Public IP.
Learning Outcomes

Docker Containerization
AWS EC2 Administration
AWS RDS Configuration
Multi-tier Application Architecture
Docker Image and Container Management
Cloud Deployment
